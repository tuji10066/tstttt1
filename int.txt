                <ul class="dot">
                    <li class="list-dot">パソコン画面、モニターの画面を第三者から見える位置に設置する。</li>
                    <li class="list-dot">外部ディスプレイ、モニターを複数接続する。<br><span class="indent">※物理的にモニターが接続してあるか、ないかにかかわらず、ディスプレイの設定画面で複数の接続がされているように表示されている場合には、受検を開始できません。</span></li>                    
                    <li class="list-dot">パソコンに外付けのハードディスクやUSBメモリなどを接続する行為。<br>※Webカメラの接続は除く。</li>
                    <li class="list-dot">Webカメラに映る範囲に検査実施に不適切なものを置く。</li>
                    <li class="list-dot">照明が暗い、もしくは逆光などで受検者の顔が認識できない環境。</li>
                </ul>
            </li>
            <li class="column-li dashed">
                <p class="list-boss">検査中の禁止行為</p>
                <ul class="dot">
                    <li class="list-dot">飲食、喫煙。
                    </li><li class="list-dot">問題を声に出して読み上げる、独り言を言う、口を覆い隠す。</li>
                    <li class="list-dot">検査実施中Webカメラ、スピーカー、マイクを切る、切断する。</li>
                    <li class="list-dot">検査実施に不要なソフトウェアやプログラム、通信機器、機材を起動、利用する。</li>
                    <li class="list-dot">検査中にWebカメラのフレームから外れる、離席、退席、退出する。</li>
                    <li class="list-dot">通信障害などで中断した場合に、監督者と接続せずに検査を実施する。</li>
                    <li class="list-dot">実施エリアに受検者以外の第三者が入室、滞在する。</li>
                    <li class="list-dot">第三者とコミュニケーションをとる、試みる（会話、電話、メール、SNSなど）。</li>
                    <li class="list-dot">受付、問題、回答などの検査情報を記録、記憶する。</li>
                    <li class="list-dot">机や壁、身体などに書き込みをする。</li>
                    <li class="list-dot">検査実施後にメモ用紙の裏と表を監督者に見せない。</li>
                </ul>
            </li>
            <li class="column-li dashed">
                <p class="list-boss">悪質な不正行為</p>
                <ul class="dot">
                    <li class="list-dot">監督者の指示に従わない、禁止事項を繰り返す。</li>
                    <li class="list-dot">本人確認書類の偽造、改造。</li>
                    <li class="list-dot">受検者本人以外が受検する、または試みる。</li>
                    <li class="list-dot">資料を見る、Webで検索するなどカンニングをする、または試みる。</li>
                    <li class="list-dot">第三者からの支援を受ける、または試みる。</li>
                    <li class="list-dot">検査内容をコピー、持出し、録画、録音、書き写し、拡散する行為。</li>
                </ul>
            </li>
        </ul>
    </div>

    <div class="h3-div" id="5-2">
        <img class="h3-no" src="images/__number_02.svg">
        <h3>その他の注意事項</h3>
    </div>
    <p class="body-text">
        その他の注意事項は以下の通りです。必ずご確認ください。
    </p>

    <div class="column column-3">
        <h5 class="margin-h5" id="5-2">その他の注意事項</h5>
        <hr class="vertical-line line-list">
        <ul class="column-ul">
            <li class="column-li dashed" id="non-dash-1">
                <ul class="dot">
                    <li class="list-dot">メモ用紙は、問題への回答にあたって必要な書き込みにのみご利用ください。<br><span class="indent"><span class="under-p">※検査終了後にメモ用紙の確認をいたしますので、監督者によく見えるようご提示ください。</span></span></li>
                    <li class="list-dot">検査実施中に各種アプリケーションなどからの通知が画面上に表示されないように、必ず事前にご自身で設定のうえで受付にお入りください。</li>
                    <li class="list-dot">受付開始直後から、カメラ映像、音声、パソコンの操作内容は監督されています。</li>
                    <li class="list-dot">検査実施中は、体調不良を含めていかなる理由でも離席、退席することは認められておりません。</li>
                    <li class="list-dot">検査実施中は、受付担当とは別の監督者からお声がけすることもございます。</li>
                    <li class="list-dot">検査中に「次へ」ボタンが表示されないなどの不具合と思われる事象が起きた場合、監督者が確認させていただきますので、その場ですぐにご申告ください。なお、当該検査の終了後にご申告いただいてもやりなおしはできませんので、ご留意ください。</li>
                </ul>
            </li>            
        </ul>
    </div>
